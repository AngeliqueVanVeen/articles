Introduction

This is an example of a website where the homepage contains articles that refer to the relevant page. The bottom of each row adapts to the article with the largest height of that row so the bottom-line is always straight.

Responsive

The website is responsive for all devices on which websites can be viewed. The number of articles that are next to each other varies per device:
1 on smartphones
2 on large smartphones and small tablets
3 on tablets
4 on laptops
5 on desktops and television screens
Furthermore, this may depend on how the display of the device is set and whether the device is in portrait or landscape position.

No plug-ins

No plug-ins were used when creating this website. As a result, the website is loaded quickly.

No JavaScript

Also, no JavaScript has been used. With JavaScript, for example, a slider or photo gallery can be added or a back-to-top button.
